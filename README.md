# Архыдея клининг — лендинг

Одностраничный сайт клининговой компании «Архыдея клининг» (Душанбе).
React + TypeScript + Vite + Tailwind CSS + Framer Motion.

## Возможности

- **Hero** — оффер, контакты, CTA «Рассчитать стоимость».
- **О компании** — преимущества и статистика.
- **Тарифы** — 3 вида уборки (поддерживающая / генеральная / после ремонта).
- **Калькулятор + квиз** — пошаговая форма заявки:
  1. Параметры уборки (площадь, тип, доп. услуги) с расчётом «Итого» в реальном времени.
  2. Детали заказа (дата/время, доступ к воде/электричеству, как попадёт клинер, комментарий).
  3. Контакты (имя, телефон, адрес) → отправка заявки.
- **Футер** — контакты, соцсети, правовая информация.
- Адаптивность mobile-first, анимации появления и интерактив.

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # сборка в dist/
npm run preview  # предпросмотр сборки
```

## Настройка цен

Все ставки и цены — в одном файле: [`src/config/pricing.ts`](src/config/pricing.ts).
Меняйте `pricePerSqm`, цены доп. услуг, минимальный заказ — логика подхватит автоматически.

Контакты компании — в [`src/config/company.ts`](src/config/company.ts).

## Отправка заявок (Telegram)

1. Скопируйте `.env.example` в `.env`.
2. Заполните:
   - `VITE_TELEGRAM_BOT_TOKEN` — токен бота от [@BotFather](https://t.me/BotFather).
   - `VITE_TELEGRAM_CHAT_ID` — id чата/группы, куда слать заявки.
3. Перезапустите `npm run dev`.

Без токена форма работает в демо-режиме: заявка не уходит, а выводится в консоль браузера (показывается экран «успех»).

> ⚠️ Прямой вызов Telegram Bot API с фронтенда раскрывает токен в браузере.
> Для продакшена вынесите отправку на сервер (будущий CRM-бэкенд) —
> точка подключения уже заложена в [`src/lib/submit.ts`](src/lib/submit.ts)
> (функция `sendToCrm` + переменная `VITE_CRM_ENDPOINT`).

## Структура

  lib/         расчёт (calc.ts), форматирование, сборка текста заявки, отправка

    quiz/      пошаговая форма (калькулятор + анкета)
